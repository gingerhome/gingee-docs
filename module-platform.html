platform

A module for GingerJS platform-specific utilities and functions. Ideally used by only platform-level apps. To use this module the app needs to be declared in the `privilegedApps` list in the ginger.json server config.
Description:
  • A module for GingerJS platform-specific utilities and functions. Ideally used by only platform-level apps. To use this module the app needs to be declared in the `privilegedApps` list in the ginger.json server config.

Description:
  • Creates a new application directory structure.
Example
